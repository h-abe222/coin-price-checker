⚙️ 監視商品の設定方法

方法1: products.json を編集

リポジトリの data/products.json ファイルを直接編集してください。

{ "gold-maple-1-2oz": { "id": 628, "url": "https://www.bullionstar.com/buy/product/gold-maple-1-2oz-various-years", "name": "Canadian Gold Maple Leaf 1/2 oz", "enabled": true } }

方法2: GitHub Secretsで環境変数を設定

  1. GitHubリポジトリの Settings → Secrets and variables → Actions
  2. 以下の変数を設定:
    • GMAIL_ADDRESS - 送信元メールアドレス
    • GMAIL_APP_PASSWORD - Gmailアプリパスワード
    • RECIPIENT_EMAIL - 通知先メールアドレス
    • CURRENCY - 通貨 (JPY/SGD/USD)
  3. Actions タブから手動実行も可能
